Playing the game:
To play the game, download the Unreal Build zip folder in releases, extract the folder and run the exe

Controls:
W - Forward
A - Backward
S - Walk Left
D - Walk Right
Space - Jump

E - Grab/Throw telekinetic object (grab when hovering over object, throw when object is pulled)
Q - Create shield debris (only on shield spawnable surfaces, e.g: the floor)
F - Activate/Deactivate levitation
Tab - Swap camera position
